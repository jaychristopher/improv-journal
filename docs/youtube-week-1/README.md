## Sequential Gates

Each step must pass acceptance criteria before moving to the next. Gates are gates: don't skip.

### Stage 1 — Strategy Validation
**Artifact:** Per-video strategy snapshot in `L1-overthinking.md` etc.
**Gate criteria:**
- [ ] Target keyword has volume ≥500 AND KD ≤30 AND `video_th` SERP feature confirmed
- [ ] Source bridge exists on site with quotable definition in first paragraph
- [ ] Persona, pillar, and cross-pillar opportunity documented
- [ ] Audio file exists at `public/audio/youtube/`

**Status:** ✅ PASS for all 3 videos (Iteration 5 scoring)

### Stage 2 — SEO Metadata Lock
**Artifact:** Title, description, tags, hashtags, chapter timestamps — all per video file.
**Gate criteria:**
- [ ] Title is ≤60 chars and contains target keyword in first 50
- [ ] Description has hook (first 100 chars keyword-rich), 5 sections per schema, UTM-tracked bridge link, complementary resource
- [ ] 12-20 tags following 6-tier priority formula
- [ ] 3-5 hashtags, primary keyword first
- [ ] Chapter timestamps with keyword-rich headings, every chapter ≥10s, first chapter at 00:00, last is CTA segment

**Output:** This file's per-video sections.

### Stage 3 — Visual Storyboard
**Artifact:** Per-chapter visual plan (slides, B-roll, on-screen text, transitions).
**Gate criteria:**
- [ ] Hook (0:00-0:30): visual matches verbal, keyword on screen within 5s
- [ ] Pattern interrupt every 30-60s (visual change)
- [ ] First CTA appears at ~0:30 (verbal + on-screen URL)
- [ ] Mid-CTA at midpoint
- [ ] End-card setup at -20s (room for 3 cards)
- [ ] Bridge URL on screen for ≥10 seconds total

**Output:** Per-video storyboard section.

### Stage 4 — Thumbnail Asset
**Artifact:** Final 1280×720 PNG.
**Gate criteria:**
- [ ] Dark slate gradient background (#0f172a → #334155)
- [ ] 1-3 word accent in red/orange (#ef4444 or #f97316)
- [ ] Pillar icon (per Section 9C of channel plan)
- [ ] Readable at sidebar size (~210×118)
- [ ] Brand-consistent with prior thumbnails

**Output:** Per-video thumbnail spec section. **Producer creates the actual PNG.**

### Stage 5 — Captions / Transcript
**Artifact:** SRT or VTT file with timing.
**Gate criteria:**
- [ ] Auto-generated by YouTube on upload (acceptable starting point)
- [ ] Manual review pass: speaker name corrections, brand name capitalization, technical terms
- [ ] Improv terminology spelled correctly (Yes And, Johnstone, Spolin, UCB, internal computation)
- [ ] Saved with the upload for accessibility + AI citation

**Output:** Caption review checklist per video.

### Stage 6 — End Card Configuration
**Artifact:** Three end cards configured in YouTube Studio.
**Gate criteria:**
- [ ] Card 1: Subscribe (top-right, -20s to end)
- [ ] Card 2: Same-pillar video link (middle-right, -15s)
- [ ] Card 3: Cross-pillar video link (bottom-right, -10s)
- [ ] All linked videos exist or are queued for the same week
- [ ] Verbal mention at -20s aligns with cards

**Output:** Per-video end card section.

### Stage 7 — Pre-Upload Checklist
**Artifact:** Producer manually verifies every item before clicking publish.
**Gate criteria:** All 15 items in Section 9I of the channel plan satisfied. See per-video file for the populated checklist.

### Stage 8 — Upload + Schedule
**Artifact:** Video published or scheduled in YouTube Studio.
**Gate criteria:**
- [ ] Scheduled for Tuesday or Thursday 9am ET
- [ ] Visible to public
- [ ] Made for kids: NO
- [ ] Category: Education
- [ ] Comments: enabled

### Stage 9 — Post-Upload Cross-Promotion (Within 24h)
**Artifact:** Cross-promo package per video.
**Gate criteria:**
- [ ] Video embedded on matching bridge page (commit code change)
- [ ] llms.txt updated with video URL
- [ ] Reddit announcement posted (use draft from per-video file)
- [ ] LinkedIn post live with link in first comment
- [ ] Twitter/X post live (optional)
- [ ] All comments on video replied to within 24h

**Output:** Per-video cross-promo section with copy-paste-ready content.

### Stage 10 — Sequence Linking (Across Videos)
**Artifact:** Each subsequent video's end cards point back to the previous one. The 3 videos form a starter playlist.
**Gate criteria:**
- [ ] L1 end-card 2 points to L23 (cross-pillar, P1 → P3)
- [ ] L23 end-card 2 points to L31 (cross-pillar, P3 → P4)
- [ ] L31 end-card 2 points to L1 (loop, P4 → P1)
- [ ] All 3 videos added to "Start Here" playlist on channel
- [ ] Playlist description references all 3 videos and the site

**Output:** Sequence linking section in this README.

## Producer Workflow

For each video, in this order:

1. Open the per-video file (e.g., `L1-overthinking.md`)
2. Copy title, description, tags, hashtags, chapters into YouTube Studio upload form
3. Create thumbnail using the spec
4. Edit video using the storyboard (audio + visuals)
5. Configure end cards (paste URLs from the file)
6. Pre-upload checklist verification
7. Schedule for the assigned day
8. After publish: copy cross-promo content, post to platforms
9. Update bridge page on site to embed the video
10. Reply to comments within 24h

**Estimated time per video: 4-6 hours of producer work.**
