## 1. Channel Architecture

### Content Pillars (5)

The channel is organized around **5 content pillars**, each mapped to a persona and a search-intent cluster. Pillars become **playlists**. Every video lives in exactly one primary pillar but can be cross-listed.

| # | Pillar | Persona | Search Intent | Content Style | Audience Goal |
|---|--------|---------|--------------|---------------|---------------|
| **P1** | Mental Game | Life-application seeker | "How do I stop X?" / "How do I be more Y?" | Reframe-first, science-backed, exercise included | Stop overthinking, be present, be confident |
| **P2** | Better Conversations | Life-application seeker | "How do I [communication skill]?" | Skill-focused, immediate practice | Listen better, talk better, navigate hard talks |
| **P3** | Team Dynamics | Team leader | "How do we [team challenge]?" | Practical, exercises for groups | Build trust, run better meetings, reduce friction |
| **P4** | Improv Practice | Beginner / Stuck improviser | "How do I [improv skill]?" | Tutorial, with examples | Learn improv, get unstuck, find game |
| **P5** | Improv Theory | Performer / Teacher / Researcher | "What is [improv concept]?" | Multi-tradition analysis, depth | Understand the system, teach better, study seriously |

### Playlist Naming Convention

Each playlist gets a clear, keyword-rich name that reads as a promise:

- **P1: "Mental Game: Improv Tools for Real Life"**
- **P2: "Better Conversations: The Improv Method"**
- **P3: "Team Dynamics: What Improv Ensembles Know"**
- **P4: "Improv Practice: From Beginner to Performer"**
- **P5: "Improv Theory: Five Traditions, One System"**

### Video Naming Convention

Pattern: `[Direct Question/Promise]: [Counterintuitive Hook]`

Examples:
- "How to Stop Overthinking: The Bandwidth Problem (What Improv Teaches)"
- "Team Building Activities That Actually Work (None Are Trust Falls)"
- "The Rules of Improv (And Why Half of Them Are Wrong)"

Rules:
- Lead with the search query (what people type)
- Add the counterintuitive twist after the colon
- 60 chars max for title visibility
- Year-stamping ONLY for "best of" content that needs annual refresh

### Visual / Thumbnail Strategy

**Style:** Dark slate gradient background (matching site brand). White serif title text + one bold accent word in red/orange.

Thumbnail templates per pillar:
- **P1 Mental Game:** Brain icon + bold word ("OVERTHINKING", "PRESENT")
- **P2 Conversations:** Speech bubble icon + bold word ("LISTEN", "WITTY")
- **P3 Team Dynamics:** Group icon + bold word ("TRUST", "MEETINGS")
- **P4 Improv Practice:** Stage/spotlight + bold word ("YES AND", "GAME")
- **P5 Improv Theory:** Map/diagram + bold word ("JOHNSTONE", "TRADITIONS")

Consistent thumbnails build channel recognition. Don't deviate.

### End Card / CTA Strategy

Every video ends with the same 30-second outro:
1. "If you found this useful, subscribe for more."
2. Show the bridge URL on screen + verbal call to read it
3. Suggest 2 related videos (1 from same playlist, 1 from adjacent playlist)
4. Mention the relevant learning path on the site (audio CTA + on-screen card)

## 3. Channel Strategy Layers

The channel runs on **three video formats** working together:

### A. Long-form (8-12 min) — The Core Asset
- One per bridge (40+ videos in inventory)
- Optimized for search + AI citation
- Voiceover-driven (we already have ElevenLabs scripts working)
- These are the "competitor review" / "evergreen pillar" videos
- Drive: search rankings, AI citations, deep engagement

### B. Shorts (under 60s) — Discovery Mechanism
- One per atom or one per exercise (170+ in inventory)
- Same insight, compressed
- "If overthinking is a bandwidth problem, here's the fix in 30 seconds"
- Drive: subscribers, channel discovery, social distribution
- Each Short links to its long-form video

### C. Tutorial / Demo (5-7 min) — Practical Value
- One per exercise (17 to start)
- Shows the exercise being done (or animated diagram)
- Title: "Mirroring Exercise: A 5-Minute Trust-Building Tool"
- Drive: shareability with teams, classroom use, embedding

**Format ratio for first 90 days:**
- Long-form: 1/week (4-12 videos)
- Tutorial: 1 every 2 weeks (2-6 videos)
- Shorts: 2-3/week (24-36 videos)

Total: ~30-50 videos in 90 days, cementing channel with both depth and discovery.

## 4. Production Pipeline (Reference)

We've already proven this works for video #1:

1. Bridge content → 1.5-2K word TTS script (single-narrator with emote tags)
2. ElevenLabs v3 generates audio (~3-7 min, costs $1-5/video)
3. Video editor: audio + slides/whiteboard/screen recording + B-roll
4. Upload with full metadata (title, description, tags, chapters, hashtags)
5. Embed in matching bridge page on site
6. Cross-promote in Reddit posts, LinkedIn, free tools
